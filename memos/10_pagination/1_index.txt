페이지네이션은 두가지 전략이 가능

1. offset 
=> limit,  orderby
=> pagesize만큼 offset이 증가
=> 아이템 위치가 정적이라는 가정
=> 구현이 쉬움, 데이터가 자주 안변하면 더 좋음
=> 페이지네이션 UI와 잘 어울림

2. cursor
=> 마지막 아이템에 커서를 둔다.
=> 페이지 넘버가 아닌 커서를 요청에 전달
=> 커서 이후의 아이템을 페칭
=> 구현이 더 복잡, 커서에 해당하는 아이템이 삭제되면 애매해짐
=> 무한스크롤, SNS 피드
